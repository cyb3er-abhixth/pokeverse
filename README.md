🟡 Pokéverse HQ

A polished and interactive web platform for Pokémon fans — featuring a full Pokédex, Ash’s Pokémon team, regional forms, special mechanics, and more.
Designed with modern UI principles, smooth animations, and complete responsiveness across all devices.

🌐 Live Demo

    👉 View on GitHub Pages:
    https://cyb3er-abhixth.github.io/pokeverse/

🖼️ Features Overview
    ⭐ Homepage
        Hero section with a yellow → red gradient and subtle Pokéball overlay
        Floating Pikachu animation
        Modern header with desktop & mobile navigation
        Quick-access buttons (Pokédex, Ash’s Team, Special Mechanics)
        Smooth transitions, hover effects & shadows
    🔍 Pokédex
        Fully interactive Pokédex with 1,025+ Pokémon
        Search by name, number, or type
        Dynamic type filtering
        Pokémon cards with type gradients, hover animations & Pokéball watermark
        Detailed modal popups with stats, abilities, flavor text & evolution chains
        Infinite scrolling for optimized loading
        Fully responsive for all devices

📦 Installation

    Clone the repo:

    git clone https://github.com/cyb3er-abhixth/pokeverse.git

    Open the project:

    cd pokeverse

    Run locally (no build needed):

    open index.html

—or just drag the file into your browser.
