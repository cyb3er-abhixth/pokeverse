🟡 Pokémon Hub

A polished and interactive web platform for Pokémon fans — featuring a full Pokédex, Ash’s Pokémon team, regional forms, special mechanics, and more.
Designed with modern UI principles, smooth animations, and complete responsiveness across all devices.
